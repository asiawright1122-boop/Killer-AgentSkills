# Bienvenue sur Killer-Skills : Des super-pouvoirs pour votre agent IA

Aujourd'hui, nous sommes ravis d'annoncer le lancement de **Killer-Skills**, le premier marketplace et registre ouvert pour les capacités des agents d'IA.

## Le défi des agents IA

Les agents d'IA (comme Claude Code, Cursor, Windsurf) sont révolutionnaires, mais leurs capacités sont souvent limitées par les outils dont ils disposent. Les développeurs devaient fréquemment écrire leurs propres serveurs MCP (Model Context Protocol) à partir de zéro pour effectuer des tâches spécifiques comme manipuler des PDF, automatiser Excel ou gérer des dépôts GitHub.

Killer-Skills change la donne.

## Qu'est-ce que Killer-Skills ?

Killer-Skills est un registre centralisé de compétences professionnelles vérifiées que votre agent IA peut découvrir, installer et utiliser instantanément.

### Caractéristiques principales

-   **Registre Universel** : Plus de 1 000 compétences vérifiées disponibles en une seule commande.
-   **Intégration MCP Profonde** : Construit sur le Model Context Protocol d'Anthropic pour assurer une interaction fluide avec les agents.
-   **Support Multi-IDE** : Fonctionne dans vos outils préférés comme Cursor, Windsurf, Claude Code, Amp et Goose.
-   **Sécurité avant tout** : Toutes les compétences sont isolées et vérifiées pour une exécution sécurisée.

## Commencez dès maintenant

Booster votre agent est simple. Exécutez simplement dans votre terminal :

```bash
# Installer le CLI
npm install -g killer-skills

# Rechercher une compétence
killer search google-maps

# Ajouter la compétence à votre agent
killer add google-maps
```

## Le futur est autonome

Nous croyons en un futur où les agents pourront "apprendre" (installer) de nouvelles capacités par eux-mêmes au moment de l'exécution. Killer-Skills fournit cette infrastructure, transformant les outils d'IA en un vaste écosystème d'intelligence connectée.

Killer-Skills est un projet piloté par la communauté. Lisez la documentation, explorez le [marketplace](https://killer-skills.com/fr/skills) et commencez à contribuer avec vos propres compétences.
