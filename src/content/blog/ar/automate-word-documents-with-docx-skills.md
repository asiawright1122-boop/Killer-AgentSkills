# روائع التوثيق: أتمتة الكتابة مع مهارة Docx

يدور عالم الأعمال حول Microsoft Word. العقود، التقارير، العروض - المستندات هي العملة الأساسية. ومع ذلك، فإن تنسيق هذه المستندات يدوياً ليس بالمهمة المفضلة لدى الكثيرين.

باستخدام مهارة **docx**، يمكنك منح وكيل الذكاء الاصطناعي الخاص بك القدرة على أتمتة إنشاء وتحرير مستندات Word الاحترافية بالكامل.

```bash
# جهز وكيلك بمهارة docx
npx killer-skills add anthropics/skills/docx
```

## ماذا يمكن لمهارة Docx أن تفعل؟

تعتمد هذه المهارة على مكتبة `python-docx` القوية وتمنح الوكيل القدرة على:

### 1. الإنشاء من الصفر
إنشاء مستندات جديدة بهياكل معقدة.
-   **عناوين هرمية**: مستندات ذات بنية تنظيمية صحيحة.
-   **تنسيق احترافي**: التحكم في الخط العريض، المائل، حجم الخط، والمحاذاة.
-   **القوائم والجداول**: قوائم نقطية أو رقمية لتقديم البيانات بشكل منظم.

### 2. أتمتة القوالب
فتح المستندات الموجودة واستبدال العناصر النائبة (placeholders) ببيانات حقيقية. مثالي لإنشاء رسائل مخصصة أو عقود بكميات كبيرة.

### 3. إدراج الصور
إدراج الرسوم البيانية أو الصور برمجياً في أماكن محددة من المستند، مع التحكم في الحجم والمحاذاة.

### 4. إدارة الأقسام
إعداد تذييلات الصفحات، وأرقام الصفحات، وفواصل الأقسام.

## حالات الاستخدام العملي

### توليد التقارير التلقائي
استخراج نتائج تحليل البيانات (مثلاً من [مهارة xlsx](https://killer-skills.com/ar/blog/mastering-excel-automation-with-xlsx-skills)) وإنشاء تقرير تحليلي مفصل تلقائياً مع عناوين ورسوم بيانية وملخصات بتنسيق Word.

### إدارة العقود
دمج النماذج القانونية القياسية مع بيانات العملاء لإنشاء عقود جديدة متقنة في لحظات.

### المراسلات الجماعية
قراءة معلومات العملاء من قاعدة البيانات وإنشاء مئات رسائل الشكر الشخصية.
